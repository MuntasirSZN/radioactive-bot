This is a bot made for my minecraft server managed through azure. Running a server 24/7 would incurs more costs, specially for beefy ones. As we had a discord server anyway, adding a discord bot with minimal amounts of monitoring and management is enough. This is implemented in python, and yes vibe-coded. It is running in production already. If you want to run this, use grep command with env as params to get the required envs (too lazy to add a example env file).
